════════════════════════════════════
  PARA USAR EL DASHBOARD (Diario)
════════════════════════════════════

  MacOS:  Doble clic en → Lanzar_Dashboard.command
  Windows: Doble clic en → Lanzar_Dashboard_Windows.bat

  El dashboard se abre automáticamente en el browser.
  URL: http://localhost:8501

  IMPORTANTE: El archivo Excel debe estar en esta misma carpeta
  con el nombre: Reporte_Formacion_CAR_TRAD.xlsx

════════════════════════════════════
  INSTALACIÓN (Solo la primera vez)
════════════════════════════════════

  REQUISITO: Tener instalado Python (Anaconda recomendado)
  Descarga gratuita: https://www.anaconda.com/download

  1. Abre esta carpeta desde tu OneDrive compartido
  2. MacOS:  Doble clic en → Instalar_y_Lanzar.command
     Windows: Doble clic en → Lanzar_Dashboard_Windows.bat
     (instala automáticamente todo lo necesario)

  Si el .command no abre en Mac:
  - Clic derecho → Abrir → Confirmar "Abrir"
  - O ir a: Preferencias del Sistema → Seguridad → Abrir igualmente

════════════════════════════════════
  CÓMO USAR LOS FILTROS
════════════════════════════════════

  Los filtros están en el panel izquierdo (sidebar).
  Selecciona uno o varios valores en cada filtro.
  Los filtros vacíos = "mostrar todo".

  Filtros disponibles:
  ┌─────────────────┬────────────────────────────────────────────┐
  │ Año             │ 2024, 2025 (o ambos para comparar)         │
  │ País            │ Rep. Dominicana, Puerto Rico, Suriname...  │
  │ Unidad Negocio  │ Caribe / Trading                           │
  │ Nivel Jerárquico│ Directive → Technical Operative            │
  │ Escuela / Área  │ Sostenibilidad, Liderazgo, Cementos...     │
  │ Modalidad       │ Presencial / Virtual                       │
  └─────────────────┴────────────────────────────────────────────┘

════════════════════════════════════
  DESCRIPCIÓN DE CADA PESTAÑA
════════════════════════════════════

  📋 Resumen Global
     KPIs principales + tabla comparativa 2024 vs 2025 + gráficos

  🌍 Por País
     Participantes y horas por operación + gráfico de barras

  📊 Por Nivel
     h/p y h/HC por nivel jerárquico + brecha formativa

  🎓 Por Escuela
     Ranking de áreas + matriz Escuela × Nivel (mapa de calor)

  👤 Por Colaborador
     Busca cualquier empleado por nombre o ID

════════════════════════════════════
  EXPORTAR A EXCEL
════════════════════════════════════

  Cada pestaña tiene un botón "⬇️ Exportar a Excel".
  Descarga la tabla actual en formato .xlsx con formato incluido.
  Los filtros activos se aplican también en la exportación.

════════════════════════════════════
  ACTUALIZAR LOS DATOS
════════════════════════════════════

  Cuando tengas datos nuevos, reemplaza el archivo:
  → Reporte_Formacion_CAR_TRAD.xlsx

  Mantén exactamente el mismo nombre y las mismas pestañas:
  - BD Consolidada
  - HeadCount

  Los datos se cargan automáticamente al refrescar el browser
  o al reiniciar el dashboard.

════════════════════════════════════
  CERRAR EL DASHBOARD
════════════════════════════════════

  MacOS:  Cierra la ventana de terminal que se abrió
  Windows: Cierra la ventana negra del .bat
  O simplemente cierra el tab del browser (el servidor
  seguirá corriendo; se detiene al cerrar la terminal).

════════════════════════════════════
  ¿PROBLEMAS?
════════════════════════════════════

  Error "command not found" o similar:
  → Usa Instalar_y_Lanzar.command (primera vez)

  El browser no abre automáticamente:
  → Escribe manualmente en el browser: http://localhost:8501

  Error de permisos en Mac con el .command:
  → Clic derecho → Abrir → Abrir de todas formas

  Datos no actualizados:
  → Presiona Ctrl+F5 (o Cmd+Shift+R en Mac) en el browser
  → O presiona "C" en el sidebar para limpiar caché
